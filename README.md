# Toy blockchain

## Dependencies

* `bash-completion`
* `build-essential`
* `cmake`
* `libboost-dev`
* `libmsgpack-cxx-dev`
* `libsodium-dev`
* `pkgconf`

## Build
``` bash
cmake -B build
cmake --build build
```
