# Toy blockchain

## Dependencies

* `bash-completion`
* `build-essential`
* `cmake`
* `libboost-url1.81-dev`
* `libmsgpack-cxx-dev`
* `libomp-dev`
* `libsodium-dev`
* `pkgconf`

## Build
``` bash
cmake -B build
cmake --build build
```
