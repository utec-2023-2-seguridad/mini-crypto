# Toy blockchain

## Dependencies

* `asciidoctor`
* `bash-completion`
* `cmake`

## Build
``` bash
git clone https://github.com/utec-2023-2-seguridad/mini-crypto
cd mini-crypto

mkdir -p build
cd build

cmake ..
make
```
